## 常见问题排查

### 后端无法启动
1. 检查 Python 版本
2. 检查依赖是否安装
3. 检查 .env 配置
4. 查看错误日志

### 前端无法访问
1. 检查 Node.js 版本
2. 检查依赖是否安装
3. 检查端口是否被占用
4. 查看浏览器控制台

### 数据库连接失败
1. 检查 DATABASE_URL 格式
2. 检查网络连接
3. 检查 Supabase 项目状态
4. 验证数据库凭证

### API 调用失败
1. 检查 Kimi API Key
2. 检查账户余额
3. 查看后端日志
4. 验证请求格式

### 图片上传失败
1. 检查文件大小
2. 检查文件格式
3. 检查目录权限
4. 查看错误信息
